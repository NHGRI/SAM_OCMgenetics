## Overview

This repository contains the scripts used to process, analyze, and visualize data for the manuscript.

The code includes:
- Admixture
- MergeImputedVariants
- GEMMA
- LAI
- RandomSampling_HypergeometricDistribution
- Nature Selection

## Data Availability

Due to ethical and privacy restrictions, individual-level data are not publicly available.

Raw data and processed summary data used to generate figures are available upon reasonable request to the corresponding author.
